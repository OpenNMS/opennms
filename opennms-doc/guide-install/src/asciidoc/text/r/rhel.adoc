[[gi-install-r-rhel]]
=== RHEL

This section describes how to install _R_ on a _RHEL_ based system.

NOTE: This description was built on _RHEL 7_ and _CentOS 7.1_.

.Install the EPEL repositories
[source, bash]
----
yum install epel-release
----

.Install R
[source, bash]
----
yum install R
----
