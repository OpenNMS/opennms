= Availability Summary Report (Default Configuration for Past 7 Days)


== Report Contents

This report provides a summary of availability across one or more surveillance categories.  One page report with average, minimum and maximum values for the following availability data for the specified surveillance category over the specified date range:

* Number of Outages
* Mean Time To Restore ("MTTR") (in hours)
* Length of Time for Each Outage (in hours)
* Percent Outage (per unit in category)
* Percent Availability (per unit in category)
//QUESTION: Need to validate explanation of the last two

This report has a pie chart showing percent available time to outage time.  It can be customized with a company logo in the header that is approximately 195px by 50px.
