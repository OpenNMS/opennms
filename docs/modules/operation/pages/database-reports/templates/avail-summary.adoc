= Availability Summary Report


== Report Contents

This report provides a summary of availability across one or more surveillance categories.
One page report with average, minimum and maximum values for the following availability data for the specified surveillance category over the specified date range:

* Number of outages
* Mean Time To Restore ("MTTR") (in hours)
* Length of time for each outage (in hours)
* Percent outage (per unit in category)
* Percent availability (per unit in category)
//QUESTION: Need to validate explanation of the last two

This report has a pie chart showing percent available time to outage time.
You can customize it with a company logo (approximately 195px by 50px) in the header.
